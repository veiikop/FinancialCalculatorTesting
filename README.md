**Финансовый калькулятор**  

> **Цель:** Освоить процесс создания тестовой документации до начала тестирования.  
> Научиться анализировать требования, проектировать тест-кейсы и разрабатывать чек-листы для реального приложения.

---

## Описание проекта

Консольное приложение **"Финансовый калькулятор"**, реализующее три модуля:

1. **Расчет кредита** 
2. **Конвертер валют** (RUB ↔ USD ↔ EUR)  
3. **Калькулятор вкладов** (с капитализацией / без)

## Модули

### 1. Расчет кредита
**Вход:**  
- Сумма кредита (руб)  
- Срок (месяцев)  
- Процентная ставка (% годовых)  

**Выход:**  
- Ежемесячный платёж  
- Общая сумма выплат  
- Переплата  

---

### 2. Конвертер валют
**Вход:**  
- Исходная валюта (RUB, USD, EUR)  
- Целевая валюта  
- Сумма  

**Выход:**  
- Результат конвертации  

**Фиксированные курсы:**  
- `USD → RUB`: 90.0  
- `EUR → RUB`: 98.5  
- `EUR → USD`: 1.09  

---

### 3. Калькулятор вкладов
**Вход:**  
- Сумма вклада  
- Срок (месяцев)  
- Ставка (% годовых)  
- Тип: с капитализацией / без  

**Выход:**  
- Доход  
- Итоговая сумма  

**Формулы:**  
- Без капитализации: `Сумма × Ставка × Срок / 12 / 100`  
- С капитализацией: `Сумма × (1 + Ставка/100/12)^Срок`
